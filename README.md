# 📚 SmartLibrary — Smart Book Management System for SRKR Engineering College

> A real problem. A real solution. Built for SRKR students and library faculty.

## 🔍 The Problem We Observed

Students at SRKR Engineering College frequently **forgot book submission deadlines**, resulting in unnecessary penalties. During exam season, students wasted precious time physically visiting the library just to check if a book was available — only to find it already borrowed.

We saw this. We built the fix.



## 💡 What SmartLibrary Does

SmartLibrary is a college library management web application that streamlines the entire book borrowing experience — for both students and library faculty.



## ✨ Key Features

### 👨‍🎓 For Students
- 🔐 **Secure Login** — Access restricted to valid SRKR College students only
- 📖 **Book Catalog** — Browse and search available books in real-time before visiting the library
- 📬 **Gmail Reminders** — Automated email alerts before the submission deadline so you never pay a penalty again
- 📥 **Borrow Requests** — Request a book directly through the app

### 🏫 For Library Faculty (Librarian)
- ✅ **Accept/Reject Book Requests** — Manage student borrow requests from a dedicated dashboard
- 🪪 **Student ID Verification** — Identify borrowing students using their college Student ID
- 📦 **Barcode Scanning** — Scan book barcodes for fast and accurate checkout
- 📊 **Book Catalog Management** — Keep the catalog updated so students always see live availability


## 🚀 How It Works
Student searches catalog → Finds available book → Sends borrow request

↓

Librarian receives request → Verifies Student ID + Book Barcode → Approves

↓

Student borrows book → Gmail reminder sent before due date → Submits on time ✅
